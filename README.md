## 🎯 About

A comprehensive **Tender Management System** with two user roles - Admin and Vendor. Manage tender creation, bidding, and selection efficiently.

## 👥 User Roles

| Role | Capabilities |
|------|-------------|
| **Admin** | Create tenders, manage vendors, select winners |
| **Vendor** | View tenders, submit bids, track status |

## ✨ Features

- 🔐 **Secure Login** - Role-based authentication
- 📝 **Tender CRUD** - Full tender management
- 💰 **Bidding System** - Submit and manage bids
- 📊 **Dashboard** - Overview and statistics

## 🛠️ Technologies

Java | JDBC | MySQL | Console UI
